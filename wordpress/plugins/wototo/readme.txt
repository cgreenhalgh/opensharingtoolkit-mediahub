== Description ==
Create simple HTML5 web apps from wordpress content (pages and posts). The web apps are intended for use on recent smart phones and tablets. The generated web apps use HTML5\'s app cache so that they can be used on a recent smart phone or tablet without an internet connection. This is part of the opensharingtoolkit project.

== Installation ==
1. Upload the directory `wototo` to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress
